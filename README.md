# reyesoft/ci

```bash
yarn add --dev reyesoft-ci
composer requiere-dev reyesoft/ci
```

## Front End

### Tools

* tslint
* sass-lint
* prettier (ts, md and json files)

### Install

`package.json`

```json
{
    "name": "My cool Reyesoft project"
    "sasslintConfig": "resources/.sass-lint.yml"
}
```
